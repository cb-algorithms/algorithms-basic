# Cutting a Rod

## Solutions

```text
// maps length to price e.g. [1, 5, 8, 9, 10]
// length 1   2   3   4   5
// price  1   5   8   9   10
p: number[];

-----------------------------------------------------------
# Solution 1
-----------------------------------------------------------
f(n, 0) = 0
f(0, l) = 0
f(n, l) = max(
    f(n - 1, l),
    l < n ? 0 : p[n - 1] + f(n, l - n)
  )

return f(p.length, p.length)
-----------------------------------------------------------

-----------------------------------------------------------
# Solution 2
-----------------------------------------------------------
f(0) = 0;
f(n) = max(p[0] + f(n - 1), p[1] + f(n - 2), ..., p[n - 1] + f(0))

return f(p.length)
-----------------------------------------------------------
```
